# Changelog

Todas as mudancas notaveis deste projeto serao documentadas aqui.
O formato segue [Keep a Changelog](https://keepachangelog.com/pt-BR/1.0.0/)
e o projeto adota [SemVer](https://semver.org/lang/pt-BR/).

## [1.0.0] - 2026-06-03
### Adicionado
- Mover de uma vez todas as abas de um dominio para a janela de um Workspace do Edge.
- Modos de match: host exato, host + subdominios e padrao custom (`*.exemplo.com`).
- Seletor visual de janela de destino no popup, com preview da contagem de abas.
- Atalho de teclado (`Alt+Shift+M`) reutilizando a ultima janela de destino salva.
- Preservacao de abas fixadas; badge no icone com a contagem de abas movidas.
- Conjunto de icones 16/32/48/128.
